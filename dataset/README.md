# Dataset

The dataset used in this project was downloaded from Roboflow Universe.

Dataset format: YOLOv8

Classes:
- Gloves
- Helmet
- No-Helmet
- No-gloves
- No-vest
- Vest

The full dataset is not included in this repository because of file size limitations.
To train the model again, download the dataset from Roboflow in YOLOv8 format and update the dataset path in train.py.
